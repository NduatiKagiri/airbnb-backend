## Getting Started

## Clone this repository

  $ git@github.com:NduatiKagiri/enumerable.git
  $ cd enumerable

## Run Project
  $ Kali Rolling WSL

### Install
  $ rbenv install 3.0.1
  $ rbenv global 3.0.1
  $ ruby -v
